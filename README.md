## Download

**Gradle**
```gradle
dependencies {
   implementation 'com.github.LowLevelSubmarine:GoogleAnswersAPI:master'
}

repositories {
    maven { url 'https://jitpack.io' }
}
```

**Maven**
```xml
<dependency>
    <groupId>com.github.LowLevelSubmarine</groupId>
    <artifactId>GoogleAnswersAPI</artifactId>
    <version>master</version>
</dependency>
```
```xml
<repositories>
  <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
  </repository>
</repositories>
```
